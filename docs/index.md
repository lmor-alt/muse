# Muse Documentation Index

## Project Overview
Muse is a music theory practice application built with React + TypeScript + Vite.

## Architecture
- `/src/exercises/` - Exercise implementations by category
- `/src/components/` - Reusable UI components (piano, staff, feedback)
- `/src/audio/` - Audio engine using Web Audio API + Salamander piano samples
- `/src/stores/` - Zustand state management
- `/src/types/` - TypeScript type definitions
- `/src/styles/` - Design system variables and global styles

## Key Files
- `.claude/MEMORIES.md` - Architectural decisions and learnings
- `src/exercises/registry.ts` - Exercise registration system
- `src/styles/variables.css` - Design system tokens
